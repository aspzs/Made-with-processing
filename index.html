Made with processing

"Made with Processing" es una serie de trabajos que utiliza la biblioteca p5.js para experimentar con código creativo y crear piezas visuales a partir de parámetros dados en el código. Estas piezas van desde simples formas geométricas hasta elementos más complejos.

En esta etapa de experimentación, se utilizan elementos visuales estáticos, shaders y se juega con el ratón de la computadora. El GIF que se encuentra al inicio consta de una serie de 100 fotogramas. El resto de las piezas son el resultado gráfico de solo una de ellas.
